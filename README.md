# Análisis de Enfermedades del Corazón - UCI Dataset

**Análisis exploratorio de datos sobre enfermedades cardíacas**  
Proyecto desarrollado en Python utilizando Pandas, NumPy y Matplotlib para analizar patrones y factores de riesgo en enfermedades del corazón.

---

## ¿Por qué lo hice?

Este proyecto es parte de mi tarea académica para aprender análisis de datos con Python. me interesa entender cómo los datos pueden ayudar a prevenir problemas de salud. Es mi primera vez haciendo un análisis completo, desde la limpieza de datos hasta las conclusiones.

---

## ¿Qué hace este proyecto?

Este análisis responde preguntas importantes como:
- ¿Qué edad tienen la mayoría de pacientes con enfermedades del corazón?
- ¿El colesterol alto está relacionado con más riesgo?
- ¿Los hombres o las mujeres tienen más riesgo?
- ¿A qué edad aumenta el riesgo de enfermedad?
